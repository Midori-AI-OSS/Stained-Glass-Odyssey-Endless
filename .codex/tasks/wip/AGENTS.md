# Work In Progress

Tasks in this folder are actively being worked on and need more development.

## Folder Organization

This folder contains category-specific subfolders for organizing work-in-progress tasks:

- **cards/** - Card implementation tasks in progress
- **chars/** - Character implementation tasks in progress
- **docs/** - Documentation tasks in progress
- **items/** - Item implementation tasks in progress
- **passives/** - Passive ability tasks in progress (see subfolders for tiers)
- **relics/** - Relic implementation tasks in progress
- **tests/** - Test-related tasks in progress

### Passive Tiers

The **passives/** folder is further organized by tier:

- **boss/** - Boss-tier passive modifier tasks
- **glitched/** - Glitched-tier passive modifier tasks
- **prime/** - Prime-tier passive modifier tasks
- **normal/** - Normal-tier (baseline) passive tasks

## Workflow

Tasks in this folder are being actively developed by coders. When a task is complete and ready for review, it should be moved to the `../review/` folder in the corresponding subfolder.
